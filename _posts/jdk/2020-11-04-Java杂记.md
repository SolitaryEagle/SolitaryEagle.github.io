# RuntimeException & Error

> https://docs.oracle.com/javase/specs/jls/se11/html/jls-11.html#jls-11.1.1

二者都是 unchecked exceptions。

RuntimeException

* 糟糕的编码和设计造成的
* 普通程序可以修复

Error

* 系统资源造成的，开发人员无法控制的
* 普通程序无法修复
