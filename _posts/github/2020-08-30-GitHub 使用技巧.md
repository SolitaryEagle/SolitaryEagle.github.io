# 下载 github 仓库中的单个文件
https://raw.githubusercontent.com/${用户名}/${仓库名}/${分支名}/${文件相对于仓库的路径}

如：wget https://raw.githubusercontent.com/SolitaryEagle/SolitaryEagle.github.io/master/download/intellij-java-google-style.xml
